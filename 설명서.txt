0. 메뉴 내 이동 및 조작법
횡스크롤이 꺼져있다면 위, 아래 방향키. 켜져있다면 왼쪽, 오른쪽 방향키로 메뉴 간 이동 가능. 
Tab 키를 눌러 현재 커서가 위치한 메뉴와 항목 개수 확인 가능.
엔터를 눌러 커서가 위치한 메뉴 실행 가능.
상황에 따라 숫자키 또는 알파벳키를 눌러 빠른 이동 가능. 
Control + F키를 눌러 메뉴 이름 검색 가능.
- 검색하고자 하는 값을 입력하고 엔터를 누르면 해당 값으로 시작하는 가장 가까이에 있는 메뉴로 점프함.
- 값 앞에 슬래시(/) 기호를 붙이면 메뉴 이름 중간에 검색하고자 하는 값이 있는지 확인함.
F3 및 Shift + F3 키를 눌러 다시 찾기 기능 수행 가능.
Control + C키를 눌러 메뉴 이름을 클립보드로 복사할 수 있음.
Alt + C키를 눌러 메뉴 인트로를 클립보드로 복사할 수 있음.
Control 키를 누른 상태에서 숫자키를 누르면 해당 번호의 메뉴로 바로 이동 가능.
Alt 키 또는 Shfit 키를 누른 채로 방향키를 눌러 각각 10, 50개 항목을 한번에 이동할 수 있음.
BGM이 나오는 메뉴에서는 페이지업, 페이지다운 키로 볼륨 조절 가능.
- 다중 선택 메뉴의 기능키
① 스페이스 키를 이용하여 커서가 위치한 항목 체크 및 체크 해제 가능. (체크된 항목은 사운드가 약간 다릅니다.)
② Control + 스페이스 키는 전체 선택, Shift + 스페이스 키는 전체 해제.
③ Alt + 스페이스 키를 눌러 범위를 지정하여 해당 범위에 있는 모든 항목을 일괄 토글 가능.
④ Control + 방향키를 눌러 선택한 항목 간 이동 가능.
- 다중 선택 메뉴의 종류
① 일반 모드: 1개 이상의 항목을 선택하면 선택한 모든 항목을 제출하는 형태.
② 스왑 모드: 일반 모드와 같으나, 다중 선택이 정렬에 특화되어 있으며, 선택 후 결과를 제출하면 다음과 같은 결과가 나타납니다.
+ 2개 항목을 선택하고 엔터: 두 항목의 위치를 맞바꿉니다.
+ 1개 항목을 선택하고 바로 옆에 있는 항목에서 엔터: 두 항목의 위치를 맞바꿉니다.
+ 1개 항목을 선택하고 2칸 이상 떨어진 항목에서 엔터: 선택한 항목을 커서와 위치한 항목의 바로 옆으로 옮깁니다.
+ 그 외에는 특별히 지정된 패턴이 없는 경우 아무것도 선택하지 않은 것과 같은 효과가 나타납니다.

1. 일반적인 편집창 활용법
① 모든 기능을 다 구현하기엔 너무 게임이 무거워지기 때문에 아주 기초적인 편집 기능만을 제공합니다. 장문의 경우 메모장 등에 적은 후 클립보드로 복사할 것을 강력 권장합니다.
+ Delete, 백스페이스 키: Delete는 커서가 위치한 글자, 백스페이스는 이전 글자를 지움. Shift + 백스페이스는 전체 삭제. 서버 사이드 편집창에서는 바로 전체 삭제로 동작합니다.
+ 왼쪽, 오른쪽 방향키: 이전 글자, 다음 글자로 이동.
+ Control + 왼쪽, 오른쪽 방향키: 이전 단어, 다음 단어로 이동.
+ Shift + 왼쪽, 오른쪽 방향키: 이전 줄, 다음 줄로 이동.
+ 위, 아래 방향키: 지금까지 입력한 글자 보기.
+ Control + 위쪽 방향키 또는 홈, Control + 아래쪽 방향키 또는 엔드: 입력한 글의 맨 처음, 끝으로 이동.
② Control + V: 클립보드에 있는 문자를 편집창에 붙여넣기. 장문의 경우 강력 권장합니다.
+ 붙여넣은 글자는 CP 949에 근거해 Parsing하기 때문에 일부 컴퓨터에서 글자 단위로 읽으면 깨짐 현상이 발생할 수도 있습니다.
③ Control + C: 현재까지 입력한 내용을 클립보드로 복사함.
④ Alt + C: 대화상자 이름을 클립보드로 복사함.
⑤ F3키를 눌러 대문자 앞에 '대문자'라고 읽을지 토글할 수 있음. 
⑥ F2키를 눌러 글자 입력 시 해당 글자를 읽게 할 것인지 토글할 수 있음. 
⑦ Shift + 엔터 키를 눌러 리턴 문자열 입력 가능.
⑧ 기본값이 존재하는 편집창에서는 F5키를 눌러 기본값 확인 가능. F6키를 눌러 클립보드로 복사할 수도 있습니다.
⑨ 다 쓴 글은 엔터 키를 눌러 전송 가능. Escape 키를 눌러 취소 가능. 

2. 캐릭터 생성 및 로그인, Password 초기화
생성: '새 계정 만들기' 메뉴 실행. 이름 엔터, 암호 엔터, 이메일 엔터 순서대로. 
- 이름 중간에 Parsing을 위해 사용되는 몇몇 특수문자나 빈 칸은 삽입할 수 없으며, 영문 대문자는 영문 소문자로 모두 치환됩니다.
암호 재설정: '암호 재설정' 메뉴 실행 후 이름 엔터, 이메일 엔터. 이후 새로운 암호를 입력하고 엔터. 
다른 계정으로 로그인: '다른 계정으로 로그인' 메뉴 실행 후 이름 엔터, 암호 엔터. 
- 위의 모든 메뉴의 작업을 완료한 후에는 엔터를 한 번 더 누르면 다시 메뉴로 빠져나옵니다. 
가장 최근에 로그인 또는 설정한 계정으로 로그인: '로그인' 메뉴 실행. 
- SR 130버전 이후부터는 '로그인' 메뉴만 실행해도 위의 기능 중 필수 기능들을 알아서 수행합니다.

3. '환경 설정' 메뉴 사용
메인 메뉴에서 열거나, 서버 접속 후 F9 키를 누르면 됨.
토글의 경우 엔터를 누르면 설정값이 바뀝니다.
굳이 건드릴 일 없는 설정은 언급하지 않았습니다. 
서버 주소 변경 및 관리: 접속할 서버 주소를 새로 등록하거나 등록된 서버 주소 중 하나를 선택하여 다음 로그인부터 해당 서버로 접속하게 합니다. 기본적으로 localhost가 등록되어 있습니다.
- 서버를 새로 등록하면 등록한 서버가 자동 선택됩니다.
버퍼 기록: 게임에서 출력된 메시지를 파일로 저장할 지 토글함. 
메뉴 횡스크롤: 메뉴 사이를 횡스크롤으로 이동할 지 토글함. 
접속 관련 메시지 듣기: 서버 접속 및 접속 해제에 관한 알림 수신 여부를 토글함.
시스템 메시지 듣기: 전체 채팅 및 다수의 불필요한 메시지 수신 여부를 토글함.
출력 메시지를 제목 표시줄에서 보기: 메시지를 스크린 리더로 읽지 않고 화면으로 읽을 지 토글함. 메뉴 이름과 같이 제목 표시줄에 출력됨.
알트 키 또는 시프트 키로 말하기 끊기: Alt나 Shift 키를 눌러 현재 읽고 있는 내용을 중간에 끊을 수 있습니다. 기본값은 켬이며, 일부 사용자는 끄는 것을 권장합니다.
오른쪽 컨트롤 키 사용: 오른쪽 콘트롤 키로 공격을 가능하게 할 지 토글함.
메뉴 항목 첫 글자 언더바 자동 제거: 켜져 있으면 메뉴 항목을 검색 중 앞에 언더바가 붙어있는 메뉴를 찾을 때 언더바를 입력할 필요가 없음.
좌우 키보드 회전 속도: 회전 키를 이용하여 좌우로 회전할 때 밀리초당 몇도씩 회전할 지 설정합니다. 소수점 단위로도 입력 가능합니다. 기본값은 0.05입니다.
상하 키보드 회전 속도: 회전 키를 이용하여 상하로 회전할 때 밀리초당 몇도씩 회전할 지 설정합니다. 소수점 단위로도 입력 가능합니다. 기본값은 0.05입니다.
좌우 마우스 회전 속도: 마우스를 이용하여 좌우로 회전할 때 얼마나 빨리 회전할 지 설정합니다. 소수점 단위로도 입력 가능합니다. 기본값은 0.1입니다. 0으로 설정하면 마우스를 비활성화합니다.
마우스 테스트: 사용 중인 마우스의 움직임을 테스트합니다. 지시 사항에 따르면 됩니다.
좌우 회전 업데이트 주기: 회전 키 또는 마우스를 이용하여 좌우로 회전할 때 몇 밀리초마다 업데이트할 지 설정합니다. 기본값은 16밀리초입니다. 주기가 짧을 수록 더 정밀하게 각도를 조절할 수 있지만, PC가 느려질 수 있습니다.
상하 회전 업데이트 주기: 회전 키를 이용하여 상하로 회전할 때 몇 밀리초마다 업데이트할 지 설정합니다. 기본값은 16밀리초입니다. 주기가 짧을 수록 더 정밀하게 각도를 조절할 수 있지만, PC가 느려질 수 있습니다.
회전 방법: 회전 방법을 선택합니다. 설정값을 어떻게 하느냐에 따라 W, S, A, D, 위, 아래, 왼쪽, 오른쪽 방향키의 사용법이 달라집니다.
시작시 사운드 재생: 게임을 실행할 시에 로고가 출력되게 할것인지, 아니면 끌 것인지에 대한 설정입니다. 켜져있으면 로고가 나오고 꺼져있으면 로고가 출력되지 않고 바로 실행됩니다. 기본값은 켜저있는 상태입니다.
메인메뉴에서 배경음악 듣기: 메인메뉴에서 음악을 들을 것인지를 토글하는 옵션입니다. 켜져있으면 음악이 나옵니다.
종료시 사운드 재생: 게임을 종료할 시에 추가로 로고를 들을 것인지 토글합니다. 켜져있으면 로고가 나옵니다.
사운드 출력 장치 변경: 게임 내에서 별도 사운드 카드를 사용할 수 있습니다. 서버 접속 중에 설정 시 접속이 자동으로 끊길 수 있습니다.
사운드 사전 사용: 사운드를 재생하면 해당 사운드를 사전에 등록하여 다음에 똑같은 사운드가 다시 재생될 때 놀라울 정도로 렉을 감소시킵니다. 그러나 메모리를 많이 잡아먹기 때문에 만약 렉이 걸리거나 메모리 용량이 부족하다면 서버 접속 중 F10 키를 눌러 초기화시킬 수 있습니다.
기본값으로 재설정: 설정을 초기화시킵니다. 초기화 중에 재실행 되지 않을 수 있습니다. 당황하지 마시고, 다시 STREET_program.exe를 찾아 실행하시면 정상적으로 재설정 되어있는 상태에서 실행될 것입니다.

5. 이동 및 회전하기 
이동 및 회전: 회전 방법에 따라 다음과 같습니다.
- 왼손으로 회전, 오른손으로 이동: 방향키로 이동, A, D로 좌우 회전, W, S로 상하 회전.
- 왼손으로 이동, 오른손으로 회전: W, S, A, D로 이동, 왼쪽, 오른쪽 방향키로 좌우 회전, 위, 아래 방향키로 상하 회전.
- 회전량 및 속도, 방법은 설정 메뉴에서 바꿀 수 있습니다.
페이지업 또는 세미콜론: 위로 이동.
페이지다운 또는 어퍼스트로피: 아래로 이동.
- 이동 및 회전 키의 특성
① 모든 이동키는 누른 채로 고정하면 해당 방향으로 계속 이동합니다.
② 회전 키는 회전 시간이 1 이상이면 누르고 있으면 해당 방향으로 계속 회전합니다.
Q, E: 각각 45도 왼쪽, 오른쪽으로 회전.
Shift + F: 직접 입력하여 좌우로 회전하는 키. 원하는 회전 각도를 입력하고 엔터를 누르면 해당 각도만큼 좌우로 회전함. 양수는 오른쪽, 음수는 왼쪽.
Shift + V: 직접 입력하여 상하로 회전하는 키. 원하는 회전 각도를 입력하고 엔터를 누르면 해당 각도로 고개를 들거나 숙임. -90부터 90 사이로 입력 가능.
스페이스: 점프하는 키. 
- 점프의 특성 
① 최대 5칸까지 가능. 
② 점프 중 고정하고 이동 시 2배의 속도로 이동 가능. 
③ 일부 무기는 점프 높이가 제한받거나 아예 점프가 불가능함.
T: 일어나고 앉는 키. 
- 앉아있기의 특성 
① 스턴이 나면 강제로 앉아야 하며 아이템 사용, 무기 사용 등이 불가능함. 
② 스턴이 아닌 경우 앉아있어도 이동이 가능하며, 느리지만 이동할 때 소리가 들리지 않음.
③ 앉은 상태에서 스페이스 키를 누르면 구르기 상태가 되며, 이때 방향키를 누르고 있으면 해당 방향으로 현재 이동속도의 4배의 속도로 구르기 가능. 일반적으로 최대 20칸까지 구르기 가능. 구르는 중에 벽에 부딪히면 약간의 대미지를 입으며, 다 구르고 나면 자동으로 일어섬.
④ 점프가 불가능한 무기를 들고 있으면 구르기도 불가능함.

6. 엔터 키의 활용 
엔터 키는 다음 용도로 사용하며 보통 같은 칸에 서면 스크린 리더가 엔터 키를 누르라고 알려주거나, 지역 추적 목록에서 잡히거나 사운드가 들립니다. 
① 포탈, 문 등에서 사용하여 연결된 존으로 이동이 가능합니다. 
② 열려 있는 상점과 같은 좌표에서 사용하면 해당 상점 품목을 엽니다. 이후 품목을 선택하고 엔터, 수량을 입력하고 엔터하면 끝. 
③ 자판기를 실행하고, 차량 내 장비 또는 스위치를 조작하거나, remote gun 재장전 등의 역할을 합니다.
④ 금고와 같은 좌표에서 사용하면 해당 금고의 비밀번호를 물어옵니다. 암호 4에서 12자리를 입력하면 금고를 열고 사용할 수 있습니다. 
⑤ 미니게임 메뉴를 열 수 있습니다.

7. 아이템 사용 
아이템을 확인하고 선택하고 사용하는 방법을 알아봅니다. 
아이템 목록 열기: I키로 가진 아이템 목록을 볼 수 있습니다.
- 아이템 목록의 특성
① 알파벳 입력을 통한 아이템 간 빠른 이동 가능. 한글 아이템 검색 시 글자 입력 후 탭 키를 눌러주면 됨.
+ 단, 숫자 키나 일부 기호 키는 기본적으로 아이템 즐겨찾기 관련 키로 배정되어 있습니다. F9 키를 눌러 '아이템 목록 내에서 즐겨찾기 키 사용' 토글을 끈 경우 숫자 키 및 일부 기호 키로 시작하는 아이템도 검색할 수 있습니다.
② 일반 메뉴와 같은 기능키를 공유함.
+ 언더바로 시작하는 아이템은 옵션에서 '메뉴 항목 첫 글자 언더바 자동 제거'를 켜서 빠르게 검색 가능합니다.
+ Control + F 및 F3 계열 키도 동작합니다.
③ 닫을 때는 Escape 키를 누르면 됨.
④ 원하는 아이템에서 엔터 키를 누르면 해당 아이템 선택이 완료되면서 메뉴를 빠져나옴.
⑤ 메뉴 내에서 다음 기능을 추가로 사용 가능함. 
+ F1: 해당 아이템의 이름을 다시 확인할 수 있음.
+ F2: 해당 아이템의 개수를 다시 확인할 수 있음.
+ 스페이스: 해당 아이템에 대한 간단한 설명을 볼 수 있음.
+ F5: 해당 아이템을 주변에 있는 다른 사람(5칸 내에 있는 사람)에게 넘김. 수량을 입력하고 엔터, 대상을 선택하고 엔터. 
+ F6: 해당 아이템을 경매에 넘김. 수량을 입력하고 엔터, 최소 입찰가를 입력하고 엔터. 
+ F7: 해당 아이템을 되팔기. 수량을 입력하고 엔터를 누르면 됨.
+ F8: 해당 아이템을 확장 아이템 즐겨찾기에 등록 및 제거함. 자세한 설명은 후술.
- 아이템 팔기에 대한 설명
① mobile_shopping_device와 shopping_drone이 필요함.
② 아이템을 팔 경우 원가의 50% 를 돌려받을 수 있음.
③ 일부 아이템은 되팔 수 없음.
- 아이템 선택 후 사용할 수 있는 기능.
① Shift + 엔터: 해당 아이템 사용. 
② Shift + Delete: 해당 아이템을 버림. 수량을 입력하고 엔터.(아이템 목록 창 내에서는 사용 불가능.)
+ 버린 후 5초정도 뒤에 소환 아이템으로 다시 나타나므로 주의해야 합니다. 
③ Alt 키를 누른 채로 특정 키 누르기: 해당 키의 슬롯에 선택한 아이템을 즐겨찾기에 등록. 
- 즐겨찾기의 특성 
① 사용 가능한 슬롯 키: 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, 하이픈, 이퀄, 백스페이스 각각 1에서 13번 아이템. 
② 한번 등록한 아이템을 다른 슬롯에 다시 등록하면 슬롯간 서로 자리를 바꿈.
③ 이미 등록된 아이템 번호에 목록에 없는 아이템을 등록하면 덮어쓰기됨.
④ 등록한 아이템은 Shift 키를 누른 채로 해당 슬롯 키를 눌러 선택 없이 사용 가능. 아이템 목록에서는 F9 키를 이용하여 '아이템 목록 내에서 즐겨찾기 키 사용' 토글을 끈 경우 사용 불가능합니다.
⑤ 아이템 목록에서는 Shift, Alt 키 등을 누르지 않고 단독으로 슬롯 키를 눌러 등록된 아이템 목록을 확인할 수 있습니다. F9 키를 이용하여 '아이템 목록 내에서 즐겨찾기 키 사용' 토글을 끈 경우 사용 불가능합니다.
⑥ Alt + 그레이브 키를 눌러 초기화 가능합니다.
- 확장 아이템 즐겨찾기: Tab 키를 눌러 호출 가능.
- 확장 아이템 즐겨찾기의 특성.
① 즐겨찾기 내의 모든 아이템 순서는 유지됩니다.
② 즐겨찾기 내에서도 기존 아이템 목록의 모든 기능키가 동작합니다.(Space 키 제외)
③ 다중 선택 메뉴 형태 중 스왑 모드로 동작하므로 메뉴 내에서 간단한 정렬이 가능합니다.

8. 무기 손에 들기, 등록, 사용, 모드 변경
- 무기의 선택 및 등록 
① 아이템 목록에서 무기를 찾아 6번에 따라 선택하면 자동으로 해당 무기를 손에 듭니다. 
② 한 번 손에 든 무기는 2번부터 순서대로 무기 목록에 자동 등록됩니다. 1번부터가 아닌 이유는 1번은 주먹으로 되어있기 때문입니다. 
③ 등록된 무기를 손에 들고자 한다면 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, 하이픈, 이퀄, 백스페이스 키를 누릅니다. 각각 1번부터 13번 무기입니다. 
④ 등록된 무기를 초기화하려면 Shift + 그레이브 키를 누르면 됨. 
⑤ 무기를 교체하면 0.5초간 움직일 수 없음.
- 무기를 손에 들었을 때의 특성 
① 점프를 할 수 없는 무기가 있습니다. 
② 일부 무기는 이동속도가 느려집니다. 그러나 일부 무기는 오히려 이동속도가 빨라집니다. 그 외에도 여러 특수한 무기가 존재합니다. 
- 무기 사용: Control 키를 누르면 선택한 무기로 공격이 가능합니다. 일부 무기는 계속 누르고 있으면 연사됩니다. 그러나 다수의 무기는 바로 공격이 안 되고 장전이 필요합니다. 
- 장전: R키로 장전 가능. 해당 장소가 안전 지역이라면 Shift + R키로 장전 해제도 가능. 
- X: 선택한 무기의 현재 상태 확인. 
- 장전 시 주의 사항
① 100발이 최대인 총에 200발의 총알을 장전할 수 없음. 100발만 장전되고 나머지는 그대로 남음. 
② 100발이 최대인 총에 총알이 70발밖에 없으면 70발만 장전됨. 
③ 50발이 최대인 총에 총알이 20발 장전되어있는데 다시 장전을 하면 30발만 추가로 장전됨. 
- 장전 해제: Shift + R키로 장전 해제 가능.
- 모드 변경: 모드 변경을 원하는 무기에서 그레이브 키를 누르면 사용 가능합니다.
- 모드 변경의 특성
① 지원되는 무기와 없는 무기가 있으며, 지원되는 무기라 하더라도 특정 액세서리가 필요할 수도 있음.
② 지원되지 않는 무기의 경우 무기 이름만 알려줌.
③ 모드 변경 간에는 딜레이가 없음.
④ 모드 변경 시 탄창을 공유하지 않을 경우 장전 해제가 먼저 이루어집니다.

9. 마우스 활용하기 
사실 위의 기능 중 일부는 마우스로도 사용 가능합니다. 
마우스를 왼쪽이나 오른쪽으로 드래그하면 해당 방향으로 회전합니다. 회전하는 속도는 상술했듯 환경 설정 메뉴에서 설정 가능합니다.
왼쪽 버튼 클릭: 공격. 
오른쪽 버튼 클릭: 장전. 
휠 클릭: 선택된 아이템 사용. 
휠 돌리기: 상하 회전.

10. 찾기 기능 
- 주변에 있는 플레이어 찾기: 기본적으로 상대가 50칸 내에 있으면 소리로 알려주나, 맵에 따라서는 정면에 있을 때만 알려주기도 합니다. 그와는 별개로 아래의 기능을 사용할 수 있습니다.
① P키를 눌러 주변에 있는 사람이나 차량을 확인할 수 있습니다. 원하는 플레이어나 차량을 선택하고 엔터 키를 누릅니다.
② 선택한 플레이어나 차량은 O키를 눌러 위치를 볼 수 있고, Shift + O키를 눌러 상세 정보를 볼 수 있습니다. 또 선택한 플레이어나 차량은 주변보기가 꺼져있더라도 소리로 알려줍니다.
③ 기본적으로 75칸까지 확인 가능하나, 무기에 따라 탐지 거리가 증가하기도 합니다.
- 추적기 활용하기: 단, 일부 맵에서는 플레이어 단독으로는 사용 불가능합니다.
① L키를 눌러 추적기를 호출합니다. 추적하고 싶은 플레이어, 차량, 로봇에서 엔터 키를 누릅니다. 가장 밑에 있는 메뉴는 실행하면 추적을 중단합니다. 그 위에 있는 메뉴는 수동 좌표 입력입니다. X, Y, Z 좌표를 입력하면 해당 좌표를 추적합니다.
② L키로 연 메뉴에서 F5, F6, F7, F8키를 눌러 플레이어, 차량, 로봇 중 하나를 선택하여 위에 출력되게 할 수 있습니다.
③ Z키를 눌러 선택한 대상 또는 좌표의 위치를 실시간으로 확인 가능. 
④ Shift + Z키를 눌러 추적기 사운드를 토글할 수 있습니다. 처음에는 사운드가 등록되어 있지 않습니다.
⑤ Alt + Z키를 눌러 추적기 사운드 관리 메뉴를 호출할 수 있음. 사운드를 등록하고 관리하는 방법은 뒤에 설명합니다.
⑥ 일부 무기를 사용하기 위해서는 추적기를 켜두어야 합니다.
- 물체 추적: K키를 누르면 주변에 있는 금고 등의 위치를 보여줍니다. 메뉴가 열리며, 원하는 목록에서 엔터를 누르면 해당 아이템을 추적 대상으로 지정할 수 있습니다.
- 지역 찾기 
① M키로 지역 목록을 연 후 원하는 지역을 선택 후 엔터. 
② N 키를 눌러 선택한 지역의 위치를 실시간으로 확인 가능. 
참고로 찾기 시 출력되는 메시지의 문법상 순서는 대략 다음과 같습니다. 
① 플레이어/아이템 등 이름 
② 방향: 시계 방향으로 알려줍니다. 추적기는 기본적으로 내 관점에서의 수평 각도와 수직 각도를 알려주지만, 환경 설정에서 '추적 정보를 절대 각도로 출력' 옵션을 끄면 역시 시계 방향으로 볼 수 있습니다.
③ 거리: 몇 칸 떨어져 있는지 알려줍니다.
④ 실제 좌표: 출력되는 것이 있고 안 되는 것이 있음. 특정 지역에서는 모든 좌표가 출력되지 않습니다.

11. 각종 정보를 확인하자 
C키를 눌러 현재 좌표와 지역을 확인할 수 있음. 일부 맵은 좌표가 안 보입니다.
F키를 눌러 현재 보고 있는 방향을 각도로 확인할 수 있음. Alt키와 함께 누르면 얼마나 고개를 들고 있는지도 알려줍니다.
H키를 눌러 현재 체력, 실드, 관통률, 가장 마지막에 나를 공격한 적이나 무기를 확인할 수 있음. 
- 각 수치의 특성 
① 체력: 기본값 10000, 최대 50000까지 회복 가능. 
② 실드: Shield 아이템을 사용하면 착용 가능. 실드가 다 깨지면 일시적으로 관통률이 증가함.
③ 관통률: 기본값 100%, 값이 높을 수록 더 세게 얻어맞음. 관통률이 아무리 높아도 일정 시간이 지나면 100%로 돌아옴. 
J키를 눌러 현재 기력 수치를 확인할 수 있음.
- 기력의 특성
① 기력이 높을 수록 이동 속도나 일부 무기를 휘두르는 속도가 더 빨라짐.
② 보통 음식을 먹으면 기력이 회복됨.
Y키를 눌러 현재 공기 수치를 확인할 수 있음. 공기 수치가 0이면 나중에 마스크를 착용할 때 불이익이 발생함. 세이프존에서 Air canister로 충전 가능. 
팝업 키를 눌러 나의 정보를 볼 수 있습니다.
- 킬, 데스, 레이팅, 경험치, 접속 IP 등을 조회할 수 있습니다.
F5키를 눌러 접속한 모든 플레이어의 목록을 볼 수 있음.
- F5 메뉴의 특성
① F1 키를 누르면 해당 플레이어의 이름을 클립보드로 복사함.
② F5 키를 누르면 해당 플레이어의 상세 상태를 볼 수 있음.
③ F6 키를 누르면 해당 플레이어를 신뢰 토글함. 신뢰하는 플레이어는 내 차량에 탑승하거나 나를 관전할 수 있으며, 언제든지 현재 위치를 조회할 수 있음.
④ F7 키를 누르면 해당 플레이어를 손절 토글함. 손절한 상대의 말은 나에게 들리지 않습니다.
⑤ F8 키를 누르면 해당 플레이어를 관전합니다. 관전모드 중일 때만 사용 가능합니다. 관전 중 C키나 H키를 누르면 해당 플레이어의 관점에서 정보를 알려줍니다. 해제하려면 채팅 명령어로 '/관전'을 입력하거나, 자신을 관전하면 됩니다.
⑥ 엔터 키를 누르면 해당 플레이어에게 비밀 채팅을 보낼 수 있습니다. 비밀 채팅에 답장을 하고 싶다면 채팅창에 '/답장'을 입력하고 메시지를 입력하십시오.

12. 이전 메시지 확인 및 채팅 
쉼표, 마침표: 이전, 다음 지난 메시지로 이동. Shift 키와 함께 누르면 맨 처음, 맨 끝 메시지로 이동함. 
대괄호 열기, 대괄호 닫기: 이전 메시지 분류, 다음 메시지 분류로 이동. 메시지 분류를 바꾸면 쉼표, 마침표 키로 보이는 내용이 분류에 맞게 필터링됨. 
- 원하는 분류에서 백슬래시 키를 누르면 해당 분류의 메시지를 받지 않게 됩니다. 다시 들으려면 백슬래시를 한번 더 누르면 됩니다.
Shift + C: 출력 중인 메시지를 클립보드에 복사. 
슬래시: 채팅 창을 엽니다. 
Shift + 슬래시: 채팅 창을 엽니다. 이곳에 입력한 메시지는 같은 맵에 있는 사람에게만 들립니다.

13. 카메라 활용하기

카메라를 누른 채로 C키를 누르면 카메라의 현재 좌표를 확인할 수 있음. 
- 카메라의 주 사용 용도 
① 낭떠러지 및 벽 감지 
② 함정 감지
- 단, 일부 지역은 카메라가 먹히지 않습니다.
카메라를 누른 채로 W, S, A, D키를 누르면 해당 방향으로 소나가 작동함.
- 소나의 특성
① 15칸 옆까지 검색 가능.
② 벽이 있는지, 낭떠러지가 있는지 등만 검색 가능하나, 카메라로 볼 수 없는 지역도 확인 가능.

14. 알아두면 좋은 개념 
여기서 정의되지 않은 아이템 사용 용도 등은 직접 타 유저에게 물어보거나 써보면서 배우기 바랍니다. 
상점: 상점에서 엔터를 누르면 메뉴가 열립니다. 여기서 원하는 아이템을 선택하고 엔터, 수량을 입력하고 엔터를 누르면 구입할 수 있습니다.
- 아이템에서 스페이스 키를 누르면 해당 아이템에 대한 간단한 설명을 볼 수 있습니다.
- 원격 상점을 이용하려면, mobile_shopping_device 아이템이 있어야 합니다. 해당 아이템을 사용하면 온라인으로 상점과 통신할 수 있습니다.
장비: 장비를 착용하면 캐릭터가 더 강해지거나, 보조 효과를 얻을 수 있습니다.
- 입는 법: 원하는 장비 아이템을 그냥 사용하면 됩니다.
- 벗는 법: U키를 누르고 벗고 싶은 장비에서 엔터를 누르면 됩니다.
함정: 일부 지역에는 함정 플래폼이 있어, 밟으면 죽거나 스턴 상태가 됩니다.
체크 포인트: 체크 포인트에 들어가게 되면 이제 해당 맵 안에서는 죽더라도 체크 포인트 위치에서 부활합니다. 또 일부 포탈은 체크 포인트로 바로 연결되기도 합니다.
독: 독에 걸리면 초마다 체력이 감소함. 치료하려면 Antibiotic 등을 먹어야 함. 
투명: Invisibility shield로 사용 가능.
- 투명의 특성
① 투명한 대상은 가까이 있어도 소리가 나지 않고, 찾기 기능으로 보이지 않음.
② 공격에 맞으면 투명이 강제로 해제됨.
순간이동: 일부 무기는 맞으면 강제로 특정 좌표로 날아감. 대표적으로 Concussive grenade가 있음. 
낭떠러지: 해당 플래폼으로 가면 밑으로 떨어지는 곳.
- 추락의 특성
① 15칸 이상 떨어질 경우 떨어진 칸 수의 500배에 달하는 대미지를 입습니다. 단, Parachute가 있으면 이 문제를 어느 정도 해결할 수 있습니다.
② 추락 중에는 점프, 앉기 등이 불가능하며, 이동 속도가 크게 느려짐.
슬라이드: 해당 플래폼 위에 있으면 일정 시간마다 자동으로 특정 방향으로 미끄러집니다.
Blade: 해당 플래폼으로 가면 체력의 절반이 날아감. 
Mine: 해당 플래폼으로 가면 더이상 당신은 이 세상 사람이 아닙니다. 여담으로 플레이어가 설치할 수 있는 Mine이 있는데, 일정량의 대미지만 줍니다. 스턴 버전으로는 Wirebomb이라는 것이 있습니다. 
Fire: 불이 난 곳에 서있으면 일정 시간마다 큰 대미지를 입습니다. 해결하려면 도망가거나 Fire extinguisher를 사용해야 합니다.
환전: 이 게임에는 두 가지의 돈이 존재합니다. Coin, Street check이 있으며, Street check 한장은 100000 Coin입니다. 
- 환전하는 방법: 세이프존 15, 15, 0에서 coin이나 street check을 사용하면 됩니다. coin에서 street check으로 환전할 때는 수수료 5% 가 붙습니다. 
- 무기의 Z 각도 조준: 회전 키 중 상하 회전 키를 이용하여 고개를 들거나 내린 뒤 발사. 단, 일부 무기는 너무 높거나 낮게 보고 있으면 무기 발사가 안 됩니다. Alt + F키를 눌러 현재 상하 회전 각도를 볼 수 있습니다.
연막: 연막 안에 들어간 플레이어, 차량, 로봇 등은 추적이 불가능함. 연막 안에 있는 플레이어, 차량, 로봇들도 마찬가지로 밖에 있는 플레이어, 로봇, 차량을 추적할 수 없음.
Remote gun: 설치 후 사용하는 총. 
- 사용 방법
① remote gun을 사서 아무데나 설치 후 _338_magnum_bullet 을 사서 엔터 키로 장전합니다. remote anti-air gun은 _battery가 필요합니다.
② remote gun controler를 구입합니다. 이후에는 직접 해보세요.
Frequency scrambler: 귀찮은 로봇, Remote gun 등으로부터 플레이어를 보호해주는 유용한 1회용 도구입니다.
미사일: 미사일은 특정 좌표에 날아가서 임무를 수행하는 무기입니다. 추적하고 있는 대상이나 좌표가 존재할 때만 사용할 수 있습니다. missile_intercept로 견제할 수 있습니다.
워프: 100~150 사이의 랜덤 칸수만큼 보고 있는 방향으로 순식간에 이동함. 
- 워프의 특성 
① 이동 시작 후에는 플레이어가 알 수 없는 좌표로 이동됨. 
② 이동 중 맵의 끝을 넘어서면 오버플로가 발생하여 반대쪽 벽으로 튕겨나감. 그리고 체력과 실드, 몇몇 버프를 모두 잃게 됨.(죽지 않음.) 
음식: 음식을  먹는 동안에는 움직일 수 없으나, 체력과 기력이 크게 회복됨.
여러 가지 패턴의 무기 소개
① 플레이어의 위치를 추적하는 무기가 있습니다.
② 플레이어의 몸에 들러붙어 점프해야 떨어지는 폭탄이 있습니다.
③ 일정 시간이 지나면 터지는 폭탄이 있습니다.
④ 일부 무기는 맞으면 무기를 손에서 놓치기 때문에 다른 무기로 바꿔야 합니다.
⑤ 일부 무기는 입힌 대미지를 체력으로 회복합니다.
⑥ 체력이 낮을 수록 강해지는 무기가 있습니다.
⑦ 일부 무기는 산탄처럼 발사되거나, 탄환이 포물선 운동을 하기 때문에 맞추기 어렵습니다.
⑧ 일부 무기는 Supressor와 함께 사용하면 사거리와 대미지가 줄어들지만 발사 소음이 많이 줄어듭니다.
⑨ 일부 무기는 계속 사용하면 열을 받으며, 너무 과열된 무기는 강제로 과열을 해소하기 위해 무기 사용이 불가능하게 됩니다. 무기의 과열율을 확인하려면 Shift + X키를 누르세요.

15. 차량 
차량 판매소에서 구입 가능. 
- 차량에 탑승하는 방법
① 원하는 좌표에 차량 아이템을 사용합니다.
② V키를 눌러 나타나는 메뉴에서 설치된 차량을 선택하고 엔터 키를 누르면 탑승 가능합니다. 허가되지 않은 남의 차는 탑승할 수 없습니다.
③ 차량을 회수하려면 가장 밑에 있는 '회수' 메뉴를 실행하면 됩니다. 단, 부서져 있는 경우 수리를 먼저 해야 합니다.
- 차량의 종류: 차량은 크게 두 가지의 형태로 나뉩니다. 하나는 외부와 격리된 밀폐형이며, 하나는 외부와 격리되지 않은 개방형입니다.
① 개방형: 외부의 소리를 모두 들을 수 있고, 특별한 제한이 없다면 기존 무기와 아이템을 그대로 사용할 수 있습니다. 그러나 1인밖에 탈 수 없고, 피격되면 일정한 확률로 플레이어가 맞을 위험이 있으며, 특히 폭발물이나 강제 순간이동에 매우 취약합니다. 또, 개방형은 탑승 중 장비 변경이 불가능합니다.
② 밀폐형: 여러 명이 동시에 탑승할 수 있으며, 모든 대미지는 차량이 대신 맞아줍니다. 그러나 전투 시에는 일부 아이템을 제외하고는 차량 내장 무기만을 사용해야 하며, 폭발음 등을 제외한 외부 소리를 들을 수 없어 위치 추적을 L키 등에 의존해야 합니다. 단, 밀폐형 차량이라도 전투 무기를 제외한 아이템 사용은 가능하며, Missile interceptor 등도 사용 가능합니다.
+ 밀폐형 차량 내에서 무기는 시동이 걸려있을 때만 사용 가능합니다. 또 일부 무기는 사용하려면 탄약이 필요하며, 해당 무기가 있는 좌표에서 X키를 눌러 탄약 정보를 확인할 수 있습니다. R키로 장전도 가능합니다.
- 차량의 특징 
① 차량이 부서지면 플레이어는 큰 대미지를 입음. 
② 차량은 Aircraft 계열을 제외하고는 공중에 뜰 수 없으며, Aircraft 계열도 일정 속도 이상으로 올라가야 뜰 수 있음. 위의 사항에 해당하지 않으면 추락하며 칸 수에 비례한 어마어마한 타격을 받게 됨. 
③ 벽이 있으면 앞으로 가지 못함. 
④ 차량이 움직이고 있을 때도 차량에서 타고 내릴 수 있음. 단 운전 중 시동을 강제로 끌 수는 없음. 
⑤ Stun grenade, Wirebomb 등을 조심해야 함. 
⑥ 수리 및 연료 공급은 기본적으로는 차량 밖에서 해야 함. 각각, Metal, Fuel tank가 필요.
⑦ 차량 내에서도 L키와 K키가 동작함. 이 기능키들은 차량 안이 아닌 밖에 보이는 것들을 알려줍니다.
⑧ 핸들을 잡은 상태에서도 Q, E키가 동작함. 단, 차량 방향을 돌리게 되면 다시 초기화되므로 주의해야 함.
- 차량 탑승 중 사용 가능한 키.
① 밀폐형 차량은 B키를 눌러 차량 밖 좌표를 볼 수 있습니다. 개방형은 그대로 C키를 사용합니다.
② V키를 눌러 현재 차량 정보를 조회할 수 있습니다.
- 차량 운전하기: 차량 핸들이 있는 좌표에서 엔터 키를 누르면 운전대를 잡을 수 있습니다. 개방형은 탑승 즉시 운전대를 잡습니다.
① Escape: 운전대 놓기, 개방형은 내리기.
② 스페이스: 시동 켜고 끄기 
③ 위, 아래 방향키, P: 각각 순서대로 엑셀, 스몰 브레이크, 빅 브레이크 밟기. 길게 누르고 있으면 계속 사용 가능. 차량이 움직이지 않고 있을 때는 아래 방향키를 누르면 180도 회전함.
④ 왼쪽, 오른쪽 방향키: 45도 회전.
⑤ Alt키를 누른채로 왼쪽, 오른쪽 방향키: 1도 단위로 회전. 정확한 회전량은 현재 속도에 비례함. 
⑥ 페이지업 또는 세미콜론, 페이지다운 또는 어퍼스트로피: 길게 누르고 있으면 계속 사용 가능합니다. 공중 차량의 경우 기수를 올리거나 내립니다. 지상 차량의 경우 주변에 계단이 있으면 타고 이동합니다.
⑦ O: 공중 차량 한정으로 기수를 0으로 조정함.
⑧ 엔터: 포탈이 있는 경우 포탈을 통과함.
⑨ A: 일부 차량에만 있는 기능. 자동 운전/운항 장치를 작동시킴. 목표 X 좌표 엔터, 목표 Y 좌표 엔터하면 해당 좌표로 자동으로 감. A키를 누르면 해제됨.
⑩ S: 일부 차량에만 있는 기능. 경적, 사이렌 등을 울릴 수 있음.
⑪ D: 현재 차량의 이동 속도와 RPM을 확인할 수 있음.
⑫ F: 현재 차량이 어느 방향을 보고 있는지 확인할 수 있음. 앞에 읽어주는 각도는 차량의 각도, 뒤에 읽어주는 각도는 플레이어의 각도입니다.
⑬ Alt + F: 현재 차량의 기수나 이동 상태를 확인할 수 있음. 뒤에 이어서 읽어주는 각도는 플레이어의 Z 각도입니다.
- 마우스로 대체 사용 가능 
① 마우스를 왼쪽이나 오른쪽으로 드래그하면 해당 방향으로 회전함. 
② 왼쪽 버튼 클릭: 엔터. 
③ 오른쪽 버튼 클릭: 빅 브레이크. 
④ 휠 클릭: 시동 켜고 끄기. 
⑤ 휠 돌리기: 엑셀, 브레이크. 
- 플레이어만 회전하기: 핸들을 잡고 있는 중에 플레이어만 해당 방향으로 회전할 수 있습니다. 단, z각도 조절 외에는 개방형 차량에서만 효과가 있습니다. 밀폐형 차량의 경우에는 핸들을 잠시 놓고 회전하면 됩니다.
- 차량 핸들을 잡고 있을때
① Alt + A, D: 플레이어만 왼쪽, 오른쪽으로 회전. 90도까지만 가능합니다.
② Alt + Q, E: 플레이어만 왼쪽, 오른쪽으로 회전. 45도씩 회전합니다.
③ Alt + W, S: 위, 아래로 회전. 공개형 차량의 경우 무기 사용이 마이너스 45도까지로 제한됩니다.

F2, F4, F7: 서버 상태 확인.

A: 아이템 개수를 입력하는 편집창에서 A를 입력하고 엔터를 누르면 현재 가지고 있는 아이템 개수로 자동 입력됨.

* 부록: 채팅 명령 
채팅창을 열고 /키를 입력 후 특정 명령을 입력하고 엔터를 누르면 사용 가능. 문법을 잘 확인하고 사용할 것.
2글자 이상의 명령어는 뒷글자를 생략되도 자동하는 경우도 있습니다.

1. 맵 관련 명령어 
/가
맵 사이를 빠르게 이동하는 기능.
- 가의 문법
① /가 단독 입력: 공개맵으로 등록된 맵 목록을 보여줍니다. 원하는 맵에서 엔터를 누르면 바로 입장할 수 있습니다. 단, 서바이벌 맵은 출력은 되지만 /서바이벌 명령어 등으로 입장해야 하고, 입장제한이 있는 맵은 플레이어에 따라 출력되지 않을 수 있습니다.
② /가 (맵이름): 해당 맵의 시작점 좌표로 이동하는 기능. 귀환금지가 있는 맵으로는 바로 갈 수 없습니다.
③ /가 (x) (y) (z) (맵이름): (맵이름)과 일치하는 이름의 맵의 지정 좌표로 이동하는 기능. 가려는 맵의 개발자 권한이 있어야 합니다. (맵이름)을 생략하면 현재 맵 내에서 이동하며, (z) 좌표까지 생략하면 좌표를 고정한 채로 x, y 좌표만 이동 가능합니다.
/소환 (이름)
플레이어를 자신이 있는 좌표로 불러냅니다. 자신이 현재 있는 맵과 대상이 현재 있는 맵의 개발자 권한이 모두 있어야 합니다.
/새맵 (맵이름) (x) (y) (기본플래폼) 
(x), (y), 크기의 (맵이름)이라는 이름을 가진 맵을 만듬. 기본 플래폼은 Z 좌표 0의 기본 플래폼을 결정하는 것인데, 아무렇게나 하면 됩니다. 맵이름 안에 콜론이나 리턴이 들어가면 안 됩니다.
/맵설정 (맵이름) (항목)
(맵이름)이라는 이름의 맵의 설정값을 바꿉니다. 맵 관리자로 등록된 플레이어만 사용할 수 있습니다. 여담으로 B키를 누르면 나타나는 메뉴에서 '명령 프롬프트를 통한 고급 설정' 항목을 선택하면 해당 명령어와 맵 이름을 자동으로 붙여줍니다.
- 2글자 이상의 설정 항목은 뒷글자를 생략해도 작동할 수 있습니다.
- 가능한 설정
① 이름: 뒤에 이름을 파라미터로 지정하면 해당 값으로 맵이름을 바꿉니다. 맵 내  포탈은 자동으로 변환되지만, 외부에서 들어오는 포탈은 자동으로 변환해주지 않으므로 수동으로 바꿔줘야 합니다.
② 공개맵: 켜져있으면 공개맵 목록에 추가되어 /가 명령어로 조회 가능합니다. 단, 귀환금지가 설정되는 경우 공개맵에서 자동으로 삭제되므로 추후 다시 등록해줘야 합니다.
③ 공개편집: 켜져있으면 모든 플레이어가 맵을 편집할 수 있으며, 꺼져있으면 맵 개설자 외에는 편집이 불가능합니다.
④ 전투: 전투를 가능하게 할지 토글하는 기능입니다. 기본값은 꺼져 있습니다.
⑤ 시작점: '/가' 명령어로 이동했을때 처음으로 나타나는 좌표를 설정합니다. 뒤에 x, y, z 파라미터를 추가로 붙여야 합니다. X나 Y 좌표를 마이너스 값으로 지정하면 랜덤으로 설정됩니다.
⑥ 개발자: 맵 제작자를 추가로 등록할 수 있습니다. 단독으로 입력하면 현재 등록된 맵 제작자의 목록을 볼 수 있고, 뒤에 플레이어 이름을 파라미터로 붙이면 해당 플레이어를 제작자 목록에서 토글합니다.
⑦ 채팅금지: 맵 채팅을 금지할 지 토글하는 기능입니다.
⑧ 순간이동금지, 귀환금지: 순간이동금지는 해당 맵에서 나가는 것을 제한하고, 귀환금지는 해당 맵에 들어오는 것을 제한합니다. 개발자는 영향을 받지 않습니다.
⑨ 입장제한: 특정 조건을 갖춘 사람만 맵에 출입할 수 있게 제한할 수 있습니다. 조건으로는 경험치, 임무, 아이템 중 하나를 지정할 수 있으며, 단독 숫자를 입력하면 경험치, 그 외에는 임무 또는 아이템으로 인식합니다. 단, 포탈 및 문에는 적용되지 않으며, 포탈 및 문에 적용하려면 각각 설정해줘야 합니다. 뒤에 파라미터를 붙이지 않고 단독으로 입력하면 현재 설정된 값을 볼 수 있습니다.
⑩ 입장, 퇴장, 이동시작, 이동끝: 각각 맵 내 입장 사운드, 맵 퇴장 사운드, 맵 내 이동 시작 사운드, 맵 내 이동 종료 사운드를 지정할 수 있습니다. 뒤에 파라미터를 붙이지 않고 단독으로 입력하면 현재 설정된 사운드를 확인할 수 있습니다.
⑪ 패닝: 1이 아닌 숫자를 입력하면 해당 맵에 들어갈 때 전체 패닝 및 볼륨을 해당 수치로 맞춰줍니다. 맵별로 사운드 환경을 다르게 하고 싶을 때 사용할 수 있습니다. 뒤에 파라미터를 붙이지 않고 단독으로 입력하면 현재 설정된 값을 볼 수 있습니다.
⑫ 터널시야: 0 이상의 숫자를 입력하면 정면 기준으로 해당 각도 밖에 있는 사람은 비컨 소리가 들리지 않음.
⑬ 부활: 3가지 모드를 토글할 수 있으며, 해당 맵에서 죽었을 때 어디에서 부활하게 할 지 결정함. 직접 해보세요.
⑭ 추적금지: 맵 내에서 추적 기능을 못 사용하게 막음. 단, 차량은 효과가 없습니다.
⑮ 투시금지, 좌표금지: 투시금지는 카메라로 벽 너머를 투시할 수 없게 할 수 있는 옵션이고, 좌표금지는 맵 내의 좌표를 볼 수 없게 하는 옵션입니다.
- 서바이벌 맵 전용 설정
① 시체금지: 죽으면 시체가 나타나는지 토글합니다.
② 시작아이템: 맵에 처음 들어오거나, 죽었을 때 주는 기본 아이템을 지정합니다.
+ 아이템 문법은 아이템 이름 세미콜론 개수 쉼표의 반복입니다.
+ 예1: 시작아이템 coin;1000000,health_drink;10
+ 예2: 시작아이템 knife;1,grenade;10,remote_gun;5
- 만든 맵을 꾸미는 법 
① B키를 누릅니다. 맵 에디터가 열립니다. 
② 만들 항목을 선택합니다. 
③ 이후는 직접 해보세요. 
- 맵 에디터의 특징 
① 플래폼, 리프트 선택 창에서는 스페이스 키, Shift + 스페이스 키, Alt + 스페이스 키를 이용하여 사운드를 들어볼 수 있음. 
② 사운드 효과, 사운드 환경 등의 사운드 목록은 일반적으로 다중 선택 메뉴입니다. 2개 이상의 항목을 선택하게 되면 해당 사운드 중 랜덤으로 재생됩니다.
③ 좌표 입력 시에는 다음과 같은 정규 표현식을 사용할 수 있음. 
+ 숫자 입력: 입력한 좌표로 설정. 음수 인식합니다.
+ 아무 값도 입력하지 않음: 현재 서있는 좌표로 설정. 
+ 플러스, 언더바, 스타, 슬래시 입력 후 숫자 입력: 현재 서있는 좌표에서 입력한 숫자만큼 더하거나 빼거나 곱하거나 나눈 좌표로 설정. 
④ 맵 이름 입력 시 아무 값도 입력하지 않고 엔터를 누르면 현재 서있는 맵으로 자동 설정됨. 
⑤ 아이템 제한이나 소환 목록 작성 시 아이템 이름 사이에 쉼표 또는 빈 칸으로 구분하여 한번에 여러 개의 아이템 등록 가능.
⑥ 아이템 제한에 카테고리를 입력하면 해당 카테고리의 모든 아이템이 자동으로 제한됨. 현재는 6개가 제공됩니다.
⑦ 아이템 소환은 등록 시 뒤에 언더바 한 개가 자동으로 붙습니다.
⑧ 문이 움직일 때 칸 수가 아닌 방향만을 지정할 때는 한글로 입력합니다. (예: 서쪽, 동, 좌, 오른쪽, 남, 앞쪽, 아래, 상 등)
+ 예제에 꼭 없더라도 어지간한 것은 다 되며, 앞 글자만 써도 인식합니다.
+ 이렇게 지정할 경우 실제로는 문 크기만큼 해당 방향으로 움직입니다.
⑨ '기존 데이터 편집' 메뉴를 실행하면 현재까지 등록한 맵 데이터 중 일부를 편집하거나 삭제할 수 있음.
+ 변경을 원하는 항목에 커서를 맞춘 후 엔터를 누르면 바로 편집 가능합니다. 단, 좌표 편집은 불가능하며, 좌표 편집은 아래의 기느 을 활용하면 됩니다. 아이템 소환 등 일부 항목은 엔터로 편집이 불가능합니다. 
+ 변경을 원하는 항목에 커서를 맞춘 후 F5키를 누르면 좌표를 입력하는 창이 나타나며, 좌표를 입력하면 해당 좌표로 요소를 이동할 수 있습니다. 같은 좌표를 입력하면 동작하지 않습니다.
+ 변경을 원하는 항목에 커서를 맞춘 후 F6키를 누르면 역시 좌표를 입력하는 창이 뜨며, 좌표를 입력하면 해당 좌표로 요소를 복사할 수 있습니다. 만약 스페이스 키로 선택한 항목이 있는 경우 해당 항목 옆으로 복사합니다.
+ 변경을 원하는 항목에 커서를 맞춘 후 F7키를 누르면 X, Y, Z 방향 중 하나로 데이터를 분할할 수 있습니다.
* 기준이 되는 좌표를 입력하면 실제로는 해당 좌표 바로 다음 좌표부터 두번째 분할 데이터가 새로 생기고, 기존 데이터는 첫번째 분할 크기만큼 줄어듭니다.
* 예를 들어 X가 200~300인 데이터를 X 축으로 225, 240, 270, 290 좌표에서 분할했다고 한다면, 200~225, 226~240, 241~270, 271~290, 291~300으로 나눠지는 것을 볼 수 있습니다.
+ 변경을 원하는 항목에 커서를 맞춘 후 F8키를 누르면 해당 데이터를 다른 맵으로 내보낼 수 있습니다. 내보내려는 맵의 개발자 권한이 있어야 합니다. 다중 선택한 상태에서 내보내는 것도 가능합니다.
+ F5, F6, F7의 좌표 기본값은 기존 데이터에 근거합니다. 만약 현재 플레이어가 서있는 좌표를 기본값으로 하고 싶다면 Shift 키와 함께 누르면 됩니다.
+ 스페이스 키를 눌러 여러 항목을 다중 선택한 뒤 엔터를 누르면 해당 항목들을 삭제할 수 있습니다. 다중 선택 메뉴의 기능키를 참조하세요.
⑩ '명령 프롬프트를 통한 고급 설정' 메뉴를 실행하면 현재 맵의 고급 설정을 맵 이름을 입력하지 않고 편리하게 할 수 있습니다. 자세한 설명은 '/맵설정' 채팅 명령 참조.
/화재진압
맵 개발자만 사용할 수 있는 명령어로, 맵 내의 모든 화재를 진압합니다.
/함정초기화
맵 개발자만 사용할 수 있는 명령어로, 맵 내의 설치된 모든 함정을 제거합니다.
/리모트건초기화
맵 개발자만 사용할 수 있는 명령어로, 맵 내의 설치된 모든 리모트건을 제거합니다.
/로봇초기화
맵 개발자만 사용할 수 있는 명령어로, 맵 내의 모든 로봇을 즉시 파괴합니다.
/맵
만든 맵 리스트를 확인함. 
/맵삭제 (맵이름) 
다음 서버 재시작 이후부터 (맵이름)에 해당하는 맵을 삭제하겠다고 예약하는 기능.

3. 미니게임 또는 이벤트
/힌트
메인 맵에서 선물의 위치를 찾음. 선물과 같은 좌표에서 엔터를 누르면 Street check을 공짜로 얻을 수 있음. 
/숫자 (값)
숫자 게임이 진행 중일 때 (값)에 해당하는 숫자를 제출함. 1~99 사이의 숫자만 가능하며, 이미 다른 사람이 제출한 숫자는 제출 불가능함. 
/깃발게임
깃발 게임의 정보를 확인함. /깃발게임 (포인트)와 같은 문법으로 입력하면 양 팀에 (포인트)의 체력이 주어지고 게임이 시작됨.
- 깃발 게임의 특성
① S flag, R flag 관련 존들을 1개 이상 설치해야 시작 가능.
② 맵 개발자만이 시작할 수 있음.
③ Flag는 한번에 1개만 들 수 있음.
④ 맵 밖으로 도망치거나 죽으면 Flag를 땅에 떨어뜨림.
⑤ Flag을 깃대에 가져가서 사용하면 상대 조직의 목숨이 감소함.
/라스트맨
라스트맨 스탠딩 게임의 정보를 확인함. /라스트맨 (목숨 개수)와 같은 문법으로 입력하면 맵 내에 있는 모든 플레이어를 참가자로 하고, 참가자들에게 (목숨 개수)의 목숨이 주어지고, 게임이 시작됨.
- 라스트맨 스탠딩의 특성
① 맵 개발자만이 시작할 수 있음.
② 목숨 개수를 음수로 입력하면 개최자는 참가자에서 제외하게 됨.
③ 참가자가 아닌 사람이나 목숨이 다한 사람이 중간에 맵 안에 들어가는 것은 맵 개발자 외에는 불가능함.
④ 참가자가 게임에서 나가면 즉시 패배함.
⑤ 참가자가 맵 밖으로 나가면 죽은 것으로 간주하여 목숨 1개를 잃음.
⑥ 최후의 생존자 1인이 승리함.
/투표
투표가 진행 중일 때 해당 투표에 참가함. 이후엔 안내되는대로 진행하면 됩니다.

5. 캐릭터 관리 
/입문자졸업
입문자 플래그를 강제로 내림. 일반적으로 강제로 내리지 않아도 50분간 유지됩니다. 
/킬데스초기화
킬, 데스 수를 초기화함. 
/금고철거
내가 설치한 사설 금고를 해체합니다. 단, 환불은 없으며, 안에 있는 모든 아이템도 날아가므로 신중하게 결정해야 합니다.
/서바이벌
서바이벌 모드를 토글함. 
- 서바이벌 모드의 특성 
① 접속을 끊어도 아이템이 저장되지 않고,않음. 죽으면 아이템이 날아가며, 시체가 생김. 
② 서바이벌 존 내에서만 움직일 수 있음. 
③ 관전모드를 사용한 시간을 제외하고 나머지 플레이 시간을 합산하여 보상을 줌.
/관전모드
캐릭터를 관전 모드 겸 자리비움 상태로 할 지 토글함. 
- 자리비움 상태의 특성 
① 공격에 맞지 않고, 공격할 수 없음. 
② 이동할 수 없음. 
③ '/관전' 명령어 사용 가능. 
/관전 (누구) 
(누구)의 플레이를 관전할 수 있음. 단독으로 입력하거나 자기 자신을 관전하면 관전 중단. 관전모드 중에만 사용 가능합니다.
/자동추적변환
L키로 추적중인 대상이 연막 안에 들어갔을때 자동으로 가까운 연막을 대신 추적하게 할지 토글함.
/체크초기화 (맵이름)
(맵이름)에 해당하는 맵의 저장된 체크 포인트를 초기화합니다. 파라미터 없이 단독 입력하면 현재 내가 있는 맵의 체크 포인트를 초기화합니다.
/경고확인
받은 경고를 확인했음을 게임 내에 선언하고 경고 메시지를 삭제함. 
/음성채팅
음성 채팅 기능을 토글함. 
- 음성채팅은 긴급하게 음성으로 말해야 할 것이 있을 때 사용하는 용도로 제작되었습니다. 실시간 개인 음성 채팅이 필요하면 보이스톡, 디스코드, 스카이프, 팀톡 등을 이용하기 바랍니다.
- 음성 채팅 기능 사용법 
① Alt + V키를 눌러 음성 녹음 시작 및 중지 가능. 최대 20초까지 녹음 가능하며, 녹음을 다시 시작할 경우 기존 녹음은 삭제됨. 
② Alt + C키를 눌러 녹음한 내용을 확인할 수 있음. 
③ Alt + X키를 눌러 녹음한 내용을 서버로 전송할 수 있음. 서버로 전송 후에는 기존 녹음 파일이 삭제되므로 Alt + C키 또는 Alt + X키를 사용하기 위해서는 다시 녹음해야 함. 
/이메일 (주소)
(주소)로 이메일을 변경함. 
/저장
내 캐릭터를 즉시 저장함.
/계정초기화
캐릭터를 초기화시킴. 모든 아이템, 차량, 금고가 날아감. 
/손절 (누구) 
(누구)라는 이름을 가진 사람을 손절 토글함. 손절된 플레이어가 하는 모든 음성, 이야기를 들을 수 없습니다. 
- 단독으로 입력하면 손절한 사람 리스트를 확인할 수 있습니다.
/신뢰 (누구) 
(누구)라는 이름을 가진 사람을 신뢰 토글함. 신뢰하는 플레이어는 나의 차량에 탑승하거나 관전할 수 있습니다.
- 단독으로 입력하면 신뢰하는 사람 리스트를 확인할 수 있습니다.

* 번외: 미니게임

1. 미니게임 하는 법
미니게임 존은 20, 0, -20에 있음.
들어가서 엔터를 누르면 메뉴가 열림.
아주 심플한 구조로 되어있습니다. 위, 아래 방향키(사이드 스크롤은 왼쪽, 오른쪽 방향키)로 이동 가능하며, 원하는 메뉴에서 엔터를 누르면 하위 메뉴로 들어가거나 해당 기능을 실행합니다.
닫는 법은 Escape 키를 누르면 됩니다.
테이블 내에 있을 때에는 '/게임말' 명령어로 테이블 내에 있는 사람끼리만 대화가 가능합니다.
테이블 내에서 게임이 진행 중이라면 메뉴 내에 게임 정보가 출력됩니다.

2. 피퍼 기능키
왼쪽, 오른쪽 방향키: 타일 간 이동.
Control + 왼쪽, 오른쪽 방향키: 스페이스 키로 선택한 타일 간 이동.
위 방향키: 모든 가진 타일 보기.
아래 방향키: 커서가 위치한 타일 보기.
스페이스 키: 타일 선택 및 해제. Control 키를 같이 누르면 전체 선택, Shift 키를 같이 누르면 전체 해제.
엔터 키: 선택한 타일 내기. 선택한 타일이 없으면 커서가 위치한 타일만 내는 것을 시도합니다.
Z: 현재까지 선택한 타일 보기.
X: 남은 개수 확인
C: 가장 최근에 나온 조합 확인
V: 현재 누구 차례인지 확인
M: 마인, 리턴
P: 패스
A: 패를 직접 선택하지 않고 커서만 맞춘 뒤 A키를 누르면 해당 타일로 시작되는 가능한 조합 리스트가 나타납니다. 단, Straight, Pair Straight는 C키로 잡힐 때만 보입니다. 원하는 조합에서 엔터를 누르면 자동으로 내집니다.
F2: 피퍼 패 뒤집어서 보기.
F1: 메뉴 호출.
여담으로 피퍼 창 내에서도 쉼표, 마침표, 슬래시 키가 먹힙니다.

* 번외: 추적기 사운드 관리하기

1. 등록
- '추적 사운드 새로 만들기' 메뉴를 엽니다.
- 범위: 첫번째 편집창에는 몇 칸 내에 적이 있을 때 소리로 알려줄 것인지 적습니다.
- 왼쪽 끝 각도, 오른쪽 끝 각도: 두번째, 세번째 편집창에는 적이 어느 각도 내에 보일 때 소리로 알려줄 것인지 적습니다. 입력한 값은 자동으로 0에서 359 사이의 값으로 변환됩니다. 예를 들어, 두번째 편집창에 -10, 세번째 편집창에 10을 입력하면 상대가 -10에서 10도 각도 사이에 보일 때 소리로 알려줍니다.
- 볼륨 크기: 네번째 편집창에 입력해야 할 것은 사운드가 출력될 때의 볼륨 크기입니다.
- 웨이브폼: 마지막으로는 사운드 타입을 선택합니다. 웨이브폼을 선택하면 목록에 등록 완료됩니다.

2. 편집 및 관리
① 커서를 밑으로 내리면 지금까지 등록한 사운드 목록을 볼 수 있습니다.
- 커서가 위치한 항목에서 F1에서 F5 키를 누르면 순서대로 범위, 왼쪽 끝 각도, 오른쪽 끝 각도, 볼륨 크기, 웨이브폼을 편집할 수 있습니다.
- Delete 키를 누르면 삭제됩니다. 다중 선택하여 여러개를 동시에 삭제하는 것도 가능합니다.
- 다중 선택 메뉴 형태 중 스왑 모드로 작동합니다. 따라서 정렬이 가능합니다.
② 사운드를 자동으로 정렬하고 싶다면 F7 키나 F8 키를 누릅니다. F7은 거리순, F8은 각도 순으로 정렬해줍니다.
③ 패닝에서 엔터를 누르면 사운드의 스테레오 패닝을 변경할 수 있습니다.

3. 슬롯 바꾸기
① 메뉴에서 숫자키를 누르면 해당 번호의 슬롯으로 바뀝니다. 여러 슬롯에 서로 다른 사운드 패턴을 추가하여 추적기의 성능을 극대화시킬 수 있습니다.
② 1개 이상의 항목이 체크되어 있을 때 슬롯을 바꾸면 선택된 사운드가 해당 슬롯으로 복사됩니다.

4. 실제 추적기 사운드에 대한 해석
① 추적기에 등록된 사운드 중 플레이어가 가장 최근에 선택한 슬롯에 있는 사운드만 재생됩니다.
② 등록된 사운드는 자동으로 위에 있는 것부터 우선순위가 정해지며, 우선순위에 따라 재생됩니다.
③ 대상이 왼쪽이나 오른쪽으로 이동하면 사운드도 왼쪽이나 오른쪽으로 이동합니다. 이때 움직이는 양은 스테레오 패닝을 어떤 값으로 설정했느냐에 따라 다릅니다.
④ 음높이는 대상의 고도를 의미합니다. 고도가 같으면 사운드는 A4(440)으로 출력됩니다. 대상이 플레이어 기준에서 정확히 아래에 있으면 A2 (110), 반대로 정확히 위에 있으면 A6 (1760)이 출력됩니다.

* 번외: 사운드 표현식

사운드 효과, 사운드 환경, 엔터로 재생 등에서 사용할 수 있는 사운드의 표현식의 작성법입니다.
파라미터 사이에는 *(스타) 기호로 구분하며, 모두 붙여 씁니다. 앞의 파라미터를 모두 입력했다면 뒤의 파라미터는 생략할 수 있습니다.
1번 파라미터에는 사운드 이름을 적습니다. 사운드 이름이 여러개일 때는 |(버티컬 바) 기호로 구분합니다. 사운드 사이에 추가 슬립이 필요할 때는 사운드 이름이 들어갈 자리에 슬래시와 숫자를 대신 입력하여 채우면 됩니다. 숫자는 밀리초로 적으면 됩니다.
2번 파라미터에는 최소 무작위 밀리초와 최대 무작위 밀리초를 적습니다. 둘 사이는 |(버티컬 바) 기호로 구분하며, 최대 무작위 밀리초는 생략할 수 있습니다.
3번 파라미터에는 반복 횟수를 적습니다. 0은 무한 반복이거나 1회 재생입니다.(사운드가 반복 재생인지에 따라 다릅니다.)
4번 파라미터에는 플래그를 적습니다.
플래그 1은 무작위 재생 여부를 정합니다. 무작위 재생이 아닐 경우 위에서부터 순차적으로 재생합니다.
플래그 2는 타이머의 판정 기준을 정합니다. 체크되어 있으면 재생이 다 된 후부터 시간을 재며, 그렇지 않다면 다음 파일을 재생할 때마다 타이머가 초기화되며 시간을 잽니다. 체크하지 않았을 때 재생 중 타이머가 다 되면 바로 다음 파일로 넘기게 됩니다.


* 번외: 사용자 지정 맵 데이터 등록하기

플래폼, 문, 사운드 효과를 사용자가 컴파일 없이 목록에 등록할 수 있습니다.
1. 먼저 mapdata 폴더 안에 있는 파일 중 하나를 엽니다.
2. 목록이 적혀있습니다. 여기에 리스트에 나타나길 원하는 이름을 삽입합니다. 이름과 이름 사이에는 Return 문자열(새문단)으로 구분합니다.
* door.txt는 문, tile.txt는 플래폼, ambience.txt는 사운드 효과입니다.
* door 이름 뒤에 =(이퀄) 문자열을 삽입하고 숫자를 삽입하면 엔진이 작동합니다. 숫자는 엔터를 눌렀을때 몇 밀리초 후에 이동을 시작할 지 정하는 파라미터입니다.
3. 등록 자체는 2번까지만 작업하면 완료되지만, 실제로 소리를 듣기 위해서는 종류별로 다음과 같은 파일을 sounds 폴더 안에 배치해야 합니다. 파일이름이라고 되어있는 부분을 등록할 사용자 지정 맵데이터 이름으로 바꾸면 됩니다.
ambience: 파일이름.ogg
tile: 파일이름step1.ogg, 파일이름step2.ogg, 파일이름step3.ogg, 파일이름step4.ogg, 파일이름step5.ogg, 파일이름land.ogg, 파일이름hardland.ogg
door: 파일이름_open.ogg, 파일이름_close.ogg, 엔진이 있는 경우 파일이름_loop.ogg
 * 커널 업데이터
sp는 시에라 13.1부터 새롭게 street program.exe를 업데이트하는 프로그램이 있습니다. 커널 업데이터를 실행시키면 삐하는 비프음이 한번 울립니다. 잠시 후, 프로그램이 커널을 업데이트합니다. 대략 10초정도 걸립니다. 10초 후, 다시 프로그램을 실행시키면 정상 실행됩니다.
주의: 이 프로그램은 street program.exe를 완전히 종료시킨 후 이용하시기 바랍니다. 그렇지 않은 경우 에러날 수 있씁니다.
다른 업데이트는 자동으로 이루어집니다. 우선 프로그램을 실행하게 되면 메인 업데이트 (예: 설명서, 변경사항, 임무)를 합니다. 로그인할때에는 사운드 업데이트를 합니다. (예: 59.ogg, musicm10.ogg등)
마지막으로 mapdata업데이트는 확장 사운드팩 적용한 분을 고려하여 설정에서 따로 메뉴를 만들어뒀으니 확인하시기 바랍니다.
